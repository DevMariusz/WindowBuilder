About the Application

This is an application designed to assist in the design and building of windows, from concept to completion.

Phase 1

Phase 1 of the application is intended to assist the salesperson:

  • Record the contact details of the customer
  • Document the customer's method of contact preferences
  • Store customer information into easily retrievable database
  • Store customer order information into same database

Phase 2

Phase 2 of the application is intended to assist the estimator:

  • Record the window style preferences of the customer
  • Document the number of windows to be built
  • Document the style of windows to be built

Phase 3

Phase 3 of the application is intended to assist the fabricator:

  • Build the windows ordered to the customer's specifications
  • Keep track of build progress and update the shop foreman and estimator

Phase 4

Phase 4 of the application is intended to assist the shop foreman:

  • Keep track of materials consumed throughout building process
  • Order new materials as required for future projects
  • Record material waste
  • Record materials most commonly consumed and suggest ordering when materials reach specified quantity
